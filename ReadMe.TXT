Moviesandbox is an Open-Source 3D realtime animation software.
It uses Codeblocks with MinGW to compile in Windows, XCode to compile in MacOSX
and Codeblocks and (i think) GCC to compile on Linux.

Just run the appropriate file for your operating system.

System Requirements:

OpenGL 2.0 compatible Graphics Card (OpenGL 2.1 recommended)
Windows XP or newer
Mac OSX 10.5 or newer (tested on Lion)
Ubuntu 11.4 (tested on 2 computers with NVidia card, both 64bit)

Some Gigabytes of memory

Please update your graphics card drivers!
